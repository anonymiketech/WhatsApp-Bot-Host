* From: "ANONYMIKETECH [noreply@anonymiketech.online](mailto:noreply@anonymiketech.online)"
* Subject: "Verify your account"
* HTML includes:

  * App name
  * 6-digit code (large bold)
  * Expiry notice (5 minutes)

Create helper:
generateCode() → 6-digit number

Store verification codes in memory:
verificationCodes[email] = {
code,
expires,
attempts
}

Endpoints:

POST /send-code

* Input: email
* Generate code
* Save with 5 min expiry
* Add cooldown (60 seconds)
* Send email

POST /verify-code

* Input: email, code
* Validate:

  * Exists
  * Not expired
  * Matches
* Mark user as verified
* Delete stored code
